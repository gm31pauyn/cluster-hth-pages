# cluster-hth-pages

## 项目简介

`cluster-hth-pages` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面未针对特定域名或网站设计，可作为通用静态页面集合使用。  
仓库结构简洁，便于快速浏览、索引或部署。

## 目录说明

```
cluster-hth-pages/
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 页面引用的资源文件（如 CSS、JS、图片等）
└── README.md        # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，按功能或主题归类。
- `assets/`：共享的样式、脚本、图片等资源，按需引用。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可单独部署或组合使用。
- 归档内容以通用性为主，不涉及具体业务或服务。

## 维护说明

- 新增页面时，请统一放在 `pages/` 目录下，并保持命名规范。
- 如需引用外部资源，请优先使用 `assets/` 下的本地文件。
- 欢迎提交 Pull Request 或 Issue，但请确保内容符合本仓库的通用归档定位。
